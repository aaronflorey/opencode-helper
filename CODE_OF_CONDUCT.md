# Code of Conduct

## Our Standard

We want this project to be a respectful, constructive place for collaboration.

Examples of behavior that help create that environment:

- being respectful in disagreements
- giving and receiving feedback in good faith
- assuming positive intent while still being clear and direct
- focusing on what is best for the project and its users

Examples of unacceptable behavior:

- harassment, discrimination, or personal attacks
- trolling or deliberately disruptive behavior
- publishing private information without permission
- sustained unconstructive hostility in issues, reviews, or discussions

## Enforcement

Project maintainers are responsible for clarifying and enforcing these standards.

They may remove, edit, or reject comments, commits, issues, or other contributions that do not align with this code of conduct.

## Reporting

If you experience or witness unacceptable behavior, contact the project maintainers privately through GitHub.

For security problems, use the process in `SECURITY.md`.
